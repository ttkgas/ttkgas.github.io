001

Chemistry Reaction Extraction

LLM research · ChEMU 2020

Chemistry patents describe reactions in prose. This project turns that prose into machine-readable structure — which strings are starting materials, reagents, solvents and products; which action verbs are reaction events; which chemical belongs to which event — using an LLM constrained by a fixed, closed schema.

The real research question isn’t “can it extract.” It’s how much of the remaining error is fixable by writing better schema definitions, versus how much genuinely requires learning from labeled data. Every experiment here exists to answer that with numbers.

002

Chia Seeds — AI Meeting Notepad

Electron · Local-first · STT

A from-scratch recreation of Granola, the AI meeting notepad: a local-first Electron app that listens to a meeting, transcribes it, and fuses the user’s own sparse notes with the machine transcript into clean structured notes.

The constraint that defined the project: exactly one runtime dependency. The WAV codec, the audio chunker, the transcript stitcher, the OAuth client, the Markdown renderer, and the SSE parser are all hand-written — because the point was to understand the primitives rather than assemble them.

004

Write Yourself a Git

Python · From scratch

A working Git implementation in pure Python — no third-party dependencies, just the standard library and a careful reading of how .git/ stores history on disk. I rebuilt the object store, the reference system, and the binary staging index by hand to stop treating version control as a black box.

With the internals understood, I’m extending past the tutorial into real Myers diff and three-way merge.
